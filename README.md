## Task Overview 
- This project creates a simple sales dashboard using Tableau to analyze sales by month, region, and category.
- The CSV dataset was imported and the Order Date was converted to a monthly format to study sales trends.
- A line chart shows sales over months, a bar chart compares sales by region, and a pie chart displays sales by category.
- These visuals were combined into a dashboard with filters for better interaction and easy understanding of sales performance.
